# Contributing

## Branch Naming Convention

Danh sách type:
- feat/ thêm tính năng
- fix/ sửa bug
- chore/ dọn dẹp, đổi config, tooling
- refactor/ thay đổi cấu trúc, không đổi hành vi
- perf/ tối ưu hiệu năng
- test/ thêm/sửa test
- docs/ tài liệu
- ci/ pipeline CI/CD
- build/ build system, build config
- hotfix/ vá gấp trên production

Format khuyến nghị: `type/short-title` (lowercase, dùng dấu - để nối từ).
Ví dụ: `feat/chat-audio-pipeline`, `fix/ota-null-url`.

Quy tắc đặt tên nhánh:
- Dạng: `type/short-title`
- Chỉ dùng chữ thường, số và dấu `-`
- Không dùng khoảng trắng hoặc ký tự đặc biệt
- `short-title` nên 3-6 từ, diễn tả đúng mục tiêu nhánh
